### 通用格式一
主题 + 媒介 + 背景 + 灯光 + 色彩 + 视角 + 镜头 + 艺术风格 + 基础设置
① 画面主题是什么？（人、物品、动物、活动…）
-主体材质是什么？（黏土、模型、玻璃... [非必要] ）
② 以什么为媒介出图？（雕塑、油画、手稿... [非必要] ）
③ 主体之外的环境？（卧室、森林、城市...）
④ 主环境打什么光？（柔和光、荧光、工作室灯光...）
⑤ 整体色调什么样？（高饱和、柔和的颜色、冷色调...）
⑥ 用什么视角出图？（全景图、透视、轴测图...）
⑦ 用什么镜头效果？（微距、特写、广角...）
⑧ 用什么风格出图？（卡通、新海诚、皮克斯...）
⑨ 基础设置（--q 2 --ar 3:4 ...）
