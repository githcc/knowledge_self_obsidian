## 安装
    ```
    git clone https://github.com/nacos-group/nacos-docker.git
    cd nacos-docker
    docker compose -f example/standalone-mysql-8.yaml up -d
    ```

## 其它
1. 将Dockerfile的`https://raw.githubusercontent.com`改为`https://raw.staticdn.net`
2. mysql的默认密码是root
