## 做法
1. 闭眼，开启背景音乐
2. 听引导，做一些相应动作，注意力集中在身体的感觉
3. 缓缓结束冥想

## 休息分类
1. 身体休息：可以是被动的（睡觉、小睡）或主动的（瑜伽、伸展运动、按摩）。
2. 精神休息：在任务和会议之间进行短暂休息，创建一种断电仪式，将工作与个人事务分开，进行冥想。
3. 创意休息：可以是自然的（欣赏日出/日落、在树林里散步）或人为的（去博物馆、体验新音乐）。
4. 情感休息：花时间独处或与能让你展现完整、真实自我的人在一起，关闭社交媒体。
